What I learned
Although this is a beginner project, it reinforced several important HTML and CSS concepts, including:

Structuring a webpage with semantic HTML.
Using Flexbox to center content both vertically and horizontally.
Applying consistent spacing with padding and margins.
Working with border-radius to create modern UI components.
Following a design specification and translating it into code.
Building a responsive layout that works across different screen sizes.
One piece of CSS I'm particularly proud of is using Flexbox to perfectly center the card:

body {
  display: flex;
  justify-content: center;
  align-items: center;
  min-height: 100vh;
}
